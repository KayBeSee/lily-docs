# Supported Devices
By using different hardware wallets from different manufacterers, Lily Wallet users remove another potential single point of failure in their digital wealth security setup.

Lily Wallet is compatable with some of the most popular hardware wallets in the bitcoin ecosystem.  To view detailed instructions for setting up a device or troubleshooting, select the device below.
